
This project illustrates usage of joins in MySQL and transforming data in JSON. 
Steps to run the application.

1. Clone the repository.
2. Create database "clearGlass" as mentioned in dbConfig.js, or change the credentials as required.
2. Do "npm i"  to install all the dependencies.  
3. Start the server by using "node server .js" command on commandline.
4. Hit the api http://localhost:5000 or use the following postman collection
  https://www.getpostman.com/collections/2ac67b8449145aa1eadf
